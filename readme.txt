Create and customize an author box to display on your WordPress posts. Built for the Genesis Framework.

== Description ==

Genesis Author Box Reloaded enhances the default author box in the <a href="https://www.engagewp.com/go/studiopress/" target="_blank">Genesis Framework</a>, and provides for more control over the author box, all without having to write any code. And for those who want to really go nuts with complex author boxes, it's extremely extendable with hooks and filters for doing just that.

== Installation ==

==== Manually ====

1. Upload the plugin to wp-content/plugins.
2. Go to your wp-admin —> Plugins -> Installed Plugins.
3. Activate.

==== Automatically: ====

1. Plugins -> Add New.
2. Upload zip file.
3. Activate.

== FAQ ==

= Can this plugin be used with any theme? =

No. Genesis Author Box Reloaded can only be used with the <a href="https://www.engagewp.com/go/studiopress/" target="_blank">Genesis Framework</a>.
